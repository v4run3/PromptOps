Deployed Models

Overview of dialogue summarization and QA models currently powering PromptOps.

BART-Large CNN (SAMSum)

Active

A heavy sequence-to-sequence bidirectional autoregressive transformer model pre-trained by Facebook and fine-tuned on the SAMSum dialogue corpus. This is the primary model used for generating highly coherent long and short summaries of multi-speaker conversations.

Transformer (Seq2Seq) 400M+ Params English philschmid/bart-large-cnn-samsum
Primary Task Dialogue Summarization
Max Token Context 1024 tokens
Architecture BART (Encoder-Decoder)

Dialogue Parser + Flan-T5

Active

A hybrid Question Answering engine. Uses a fast deterministic RegEx dialogue parser combined with semantic keyword matching for highly accurate 'Who' questions based on turn order. Falls back to Google's flan-t5-base generative model for complex reasoning and reading comprehension.

Python Heuristics Flan-T5 (250M Params) Zero-Shot QA
Primary Task Question Answering
Model Engine google/flan-t5-base
Avg. Latency (GPU) ~850ms

Custom BERT-Seq2Seq

Development

The custom-built Transformer architecture (Phase 1-3). Uses a pretrained BERT encoder with a custom 6-layer decoder trained from scratch on academic dialogue datasets. Currently used for research and evaluation comparisons against state-of-the-art models.

PyTorch d_model: 512, 8 Heads Custom Arch
Primary Task Summarization & Eval
Vocabulary Size 30,522 (BERT)
Status Checkpoint best_model.pt (Colab)
